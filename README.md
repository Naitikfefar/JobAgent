# JobCopilot 🤖
AI-powered job search assistant for Indian students and professionals.

## Features
- Multi-platform job search (Internshala, LinkedIn, Indeed, Naukri, Glassdoor)
- AI match scoring (0-100%)
- AI-generated cover letters (Groq LLaMA)
- Tailored PDF resume per job
- Interview preparation AI
- Application tracker
- Telegram notifications

## Tech Stack
Frontend: React + TypeScript + TailwindCSS
Backend: Node.js + Express + MongoDB
AI: Groq (LLaMA 3.3 70B)
Agents: Python (BeautifulSoup, ReportLab)
